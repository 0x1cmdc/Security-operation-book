# ThreatHunting-book

## 简介

Threathunting-book wiki已停止维护

Threathunting-book目前已覆盖54个TID，162个场景.

主要涉及ATT&CK技术、场景测试、检测所需数据源、检测思路等。以AD域渗透技巧为中心+ATT&CK技术理解为主，如果你也对AD域渗透和ATT&CK技术有兴趣，可以私信加入我们。我们可以互相探讨学习！

最后，提醒大家，注意劳逸结合，关注个人身体健康！
