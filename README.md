# ThreatHunting-book

## 简介

Threathunting-book目前已覆盖80个TID，236个场景.

主要以AD域渗透技巧为中心+ATT&CK技术理解为主，可能涉及ATT&CK技术、模拟测试、检测思路、检测所需数据源等。

最后，提醒大家，注意劳逸结合，关注个人身体健康！
